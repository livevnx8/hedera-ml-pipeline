# Support

Use GitHub issues for reproducible bugs, documentation problems, feature requests, and example requests.

## Good Issue Reports

Please include:

- Python version
- Operating system
- Command that failed
- Full error output
- Whether the issue requires live Mirror Node access

## Security Issues

Do not post private keys, wallet files, API keys, `.env` contents, account secrets, or production endpoint credentials in public issues.

For sensitive security concerns, contact the maintainer privately before public disclosure.

## Scope

Supported:

- Public Mirror Node reads
- Library installation
- Public API usage
- Examples and tests
- Documentation improvements

Not supported in this public repository:

- Private QVX/Veda runtime internals
- Wallet custody or signing flows
- Exchange credentials
- Production trading deployment support
- Guaranteed trading performance
