# Security

This repository is a research and developer tooling project. It does not require private keys for the included demo.

## Supported Scope

Security review in this repository focuses on public-data tooling:

- Public Hedera Mirror Node reads
- Local examples and tests
- Documentation and package installation
- Public API boundaries

## Secrets

Do not commit private keys, account secrets, wallet files, `.env` files, exchange API keys, or production endpoint credentials.

## Network Calls

The demo reads public Hedera Mirror Node endpoints. Review code before connecting it to private systems.

## Financial Risk

This project does not provide financial advice or guarantee trading performance. Treat all strategy and execution examples as research infrastructure.

## Reporting Issues

Open a GitHub issue for reproducible bugs. For sensitive security issues, contact the maintainer privately before public disclosure.

Do not include private keys, account secrets, wallet files, `.env` contents, API keys, or production endpoint credentials in public issues.
